main module

main.main()[source]

The main function for running the Train Reservation System.

This function performs the following tasks: 1. Generates a schedule using ScheduleMaker. 2. Reads and modifies the schedule list as needed. 3. Saves the schedule to a file. 4. Generates train objects using TrainsMaker. 5. Saves the train objects to a file. 6. Creates a QApplication for the GUI. 7. Initializes and shows the main GUI window. 8. Runs the QApplication event loop.
